## Greysound Pro 0.1.11

- Fixed dragging local audio files onto the timeline, including WAV, AIFF, FLAC, MP3, OGG, M4A, AAC, and WMA files.
- Fixed Bounce Mix so it shows real render progress, supports longer sessions, renders directly to a local WAV file, and opens the system Save As dialog when the mix is ready.
- Hid the standalone Greysound FX Designer VST3 from Greysound Pro because the same effect is already bundled with the app.

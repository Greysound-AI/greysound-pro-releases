## Greysound Pro 0.1.7

- Fixed third-party AU and VST3 settings being lost after reopening a session by saving and restoring each plugin's complete native state, including presets, programs, modulation or routing data, bus layouts, loaded assets, and parameters beyond the first 128 exposed controls.
- Changes made directly inside a native plugin editor now mark the session dirty and trigger autosave without sending large plugin-state payloads during routine UI updates.
- Preserved complete third-party plugin state through insert and track deletion undo/redo, batch actions, insert copies, and Library channel-strip snapshots.
- Kept older sessions compatible by falling back to their saved parameter maps when no native plugin-state snapshot is available.

## Greysound Pro 0.1.10

- Improved third-party instrument and effect loading by trying compatible installed plug-in formats when the requested variant cannot create an audio processor.
- Added an actionable plug-in load failure dialog with plain-language recovery steps, format attempt results, direct access to Plug-in Settings, and copyable debug logs.
- Fixed piano-roll audition notes so they stop when the key is released, support dragging across keys, play through the track's active instrument, and use a built-in piano-like preview when no instrument is loaded.
- Fixed MIDI editor right-click handling across the piano-roll canvas, including empty areas within and between regions, so Greysound's note and region menus appear instead of the system Reload/Inspect menu.
- Fixed Show in Finder/File Explorer for local desktop sessions.
- Made Virtual MIDI Controller keys take priority over conflicting global shortcuts, including the K metronome shortcut, while the controller is open.
- Improved Credits Activity refresh controls with clearer loading, completion, and last-refreshed status.

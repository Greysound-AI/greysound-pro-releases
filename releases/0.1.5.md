## Greysound Pro 0.1.5

- Fixed desktop-local sessions appearing in the Cloud tab after chats or generated audio created cloud metadata for them.
- Cloud sessions are now shown only when they contain a downloadable project file, so previously misclassified local-session records are hidden while current `.gss` and legacy `session.json` projects remain available.
- Cloud-backed desktop sync now records its project classification before upload, keeping legitimate synced sessions separate from local AI metadata.

## Greysound Pro 0.1.3

- Fixed virtual MIDI keyboard monitoring and recording on desktop, including support for built-in and third-party instruments.
- Fixed recorded MIDI notes not appearing during recording and MIDI regions disappearing when recording stopped.
- Fixed a crash when Greysound's built-in instruments were destroyed after recording or session changes.
- Fixed empty MIDI-region creation from the timeline, piano roll, and AI tools on desktop.
- Added backwards compatibility for legacy `session.json` projects. Opening or cloning one now creates the current `.gss` structure without deleting the original legacy files.
- Fixed update prompts on macOS so updates can be installed now or deferred until the next launch, matching Windows and Linux.
- Improved update error reporting with copyable debug data, and fixed Linux AppImage update detection and version handling.
- Fixed the DAW window remaining at the session picker size or staying non-resizable on some Linux X11 desktops, including XFCE.
- Fixed startup crashes with proprietary NVIDIA drivers caused by WebKitGTK's GBM renderer.
- Added desktop crash and freeze diagnostics to help investigate user-reported failures.
